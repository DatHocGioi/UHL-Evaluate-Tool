# 🎓 UHL Course Evaluation Tool

Công cụ hỗ trợ duyệt môn và tự động thực hiện đánh giá học phần trên hệ thống Đại học Hạ Long (UHL).

## ✨ Tính năng

### 📚 Duyệt môn đánh giá

* Tự động quét danh sách môn học chưa đánh giá.
* Hiển thị đầy đủ các môn khả dụng.
* Chuyển nhanh giữa các môn học.
* Tự động tải biểu mẫu đánh giá của môn được chọn.

### 🔍 Phân tích biểu mẫu đánh giá

* Tự động quét và phân loại câu hỏi.
* Hỗ trợ 3 nhóm câu hỏi:

  * Câu hỏi trắc nghiệm
  * Câu hỏi đặc biệt
  * Câu hỏi tự luận
* Thống kê số lượng câu hỏi theo từng nhóm.

### ⚙️ Điền đánh giá tự động

* Tùy chỉnh tỷ lệ phân phối điểm đánh giá.
* Hỗ trợ cấu hình riêng cho từng mức điểm.
* Tự động xử lý câu hỏi trắc nghiệm.
* Hỗ trợ xử lý câu hỏi đặc biệt.
* Tự động sinh nội dung tự luận.

### 💾 Quản lý Profile

* Lưu cấu hình đánh giá thành Profile.
* Nhân bản và tái sử dụng Profile.
* Chuyển đổi nhanh giữa các cấu hình khác nhau.

### 🚀 Tự động hóa toàn bộ quy trình

* Quét danh sách môn học.
* Chọn môn tiếp theo.
* Phân tích biểu mẫu đánh giá.
* Điền đánh giá tự động.
* Hỗ trợ thực hiện liên tục nhiều môn.

## 📥 Cài đặt

### 1. Cài đặt Tampermonkey

Cài đặt tiện ích Tampermonkey cho trình duyệt:

* Chrome
* Edge
* Firefox
* Brave

### 2. Cài đặt UserScript

Nhấn vào liên kết dưới đây để cài đặt:

👉 https://github.com/DatHocGioi/uhl-survey-tool/raw/main/final.user.js

### 3. Sử dụng

1. Truy cập hệ thống đánh giá học phần của UHL.
2. Mở giao diện Tool.
3. Quét danh sách môn học.
4. Cấu hình Profile đánh giá.
5. Thực hiện đánh giá tự động.
